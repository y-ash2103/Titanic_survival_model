# 🚢 Titanic Survival Prediction Model

## 📌 Project Overview
This project uses the Titanic dataset to predict whether a passenger survived or not based on various features such as age, gender, passenger class, fare, and embarkation point.

It is a beginner-friendly machine learning project demonstrating data preprocessing, visualization, model building, and evaluation.

---

## 🎯 Objectives
- Perform data preprocessing and cleaning
- Visualize key patterns in the dataset
- Build a classification model using Logistic Regression
---

## 📂 Dataset
- File: `titanic_sample.csv`
- Features include:
  - PassengerId
  - Survived (Target Variable)
  - Pclass
  - Sex
  - Age
  - Fare
  - Embarked

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🔄 Workflow

### 1. Data Preprocessing
- Removed unnecessary columns
- Handled missing values
- Encoded categorical variables

### 2. Data Visualization
- Survival Count
- Survival by Gender
- Age Distribution

### 3. Model Building
- Logistic Regression

### 4. Evaluation
- Confusion Matrix
- Accuracy Score
